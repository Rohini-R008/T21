# Food Services Web App

## Project Overview

This project is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to explore services, interact with the interface, and experience smooth navigation effects.

The project structure and styling are inspired by a modern UI layout with animations and responsive design.


## Features

* Responsive Navigation Bar
* Sticky Navbar on Scroll (using Waypoints)
* Smooth Scrolling between sections
* Hero Section with call-to-action buttons
* Services and Features Section
* Image Showcase Section
* Steps / How it Works Section
* Cities Section
* Testimonials Section
* Pricing Plans Section
* Contact Form
* Footer with social links

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

* index.html → Main webpage 
* style.css → Main styling file
* script.js → JavaScript functionality
* vendors/ → External libraries (normalize, grid, animate, etc.)
* resources/ → Images, CSS, JS

## How to Run the Project

1. Download or extract the project folder
2. Ensure all folders (resources, vendors) are present
3. Open `index.html` in a web browser

## Functionality

* Navbar becomes sticky when scrolling down
* Clicking buttons scrolls smoothly to sections
* Animations appear when sections enter viewport
* Mobile-friendly layout using responsive design
